# Hive Status

Report the status of active workflow epics.

**Input:** `$ARGUMENTS` optionally contains an epic ID to filter.

**Instructions:** Read `hive/ORCHESTRATOR.md` and follow the **Status Command** section. Episodes at `state/episodes/`, epics at `state/epics/`.
